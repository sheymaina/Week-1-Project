 Africa Quiz

A simple command-line quiz game built in Python. It asks multiple-choice questions about Africa and tracks your total score.

 Features
- Multiple choice format: Questions are presented with A/B/C/D options
- Score tracking: Keeps count of correct answers
- Beginner-friendly: Straightforward Python code, great for learning loops, lists, and conditionals

 How to Run
1. Clone or download this repository
2. Open a terminal in the project folder
3. Run the script:
   python project.py
4. Enter your answer as A, B, C, or D when prompted

  How It Works
1. Questions are stored as a list of dictionaries with question, choices, and answer keys
2. The program loops through each question, prints the choices, and takes user input
3. It checks the answer against correct_ans and updates the score if correct
4. Final score is displayed at the end
